## Project Name:
Fingerprint Based Door Lock System

## Project Scope
The Fingerprint Based Door Lock System is an embedded security project that uses biometric authentication to control access.Instead of traditional keys or passwords,the system verifies a users fingerprint before unlocking the door.This project demonstrates the use of fingerprint sensing technology with a microcontroller to provide a secure,reliable and user-friendly access control system suitable for homes,offices.laboratories,and other restricted areas.

## Project Description
This project is developed using an Arduino Uno and a fingerprint ssensor.Authorized fingerprints are first enrolled into the sensors memory.Whenever a user places their fingerprint on the sensor,the system captures the fingerprint and compares it with the stored database.If the fingerprint matches an authorized user,the arduino activates a servo motor to unlock the door and displays an access granted message.After a short delay,the servo returns to its original position to lock the door automatically.If the fingerprint does not match,access is denied and the door remains locked.LEDs,a buzzer,or an LCD can also be used to indicate the system status.This project provides a secure and efficient alternative to conventional locking systems by eliminating the need for physical keys.
